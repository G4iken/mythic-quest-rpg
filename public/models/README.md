# Optional GLB/GLTF character model slots

Drop optimized `.glb` files here to replace the built-in procedural characters.
Recommended names:

- `wanderer.glb`
- `nyx.glb`
- `lyra.glb`
- `borin.glb`
- `kaida.glb`
- `zeph.glb`

Use Mixamo or Blender to export idle/run/attack clips. Keep files small for mobile, ideally under 3 MB each.
The current build includes the model-loader slot and safe procedural fallback so the game runs even without GLB files.
